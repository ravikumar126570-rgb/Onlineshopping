# 🛒 Online Shopping (Microflip Shop)

A full-stack e-commerce web application where users can browse products, add to cart, and manage authentication securely.

---

## 🚀 Features

- User Signup & Login (JWT Authentication)
- Product Listing & Details
- Add to Cart functionality
- REST API Backend
- Fast frontend using React + Vite
- Secure with Helmet & CORS

---

## 🏗️ Tech Stack

### Frontend:
- React.js
- Vite
- Context API
- CSS

### Backend:
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT
